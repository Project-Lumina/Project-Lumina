Releases

  • v1.2.1

    This release contains quite a few new features and a lot of bug fixes and improvements.

    Changelog:

    • Added support for the new website and API
    • Added the ability to download packs as zips
    • Added a better shortcut setup which includes a TOS that must be agreed too
    • Added back automatic pack downloads (you no longer have to manually download!)
    • Fixed "The file doesn't exist" bug by forcing the user to set their Minecraft folder
    • Fixed some options in the Options Editor that weren't working
    • Removed No UI and replaced it with Freelook in UI Customization
    • Other small bug fixes and improvements
    Download

  • v1.2.0.5

    This is one of the biggest releases since the initial release of the shortcut. Every single thing that was added, changed, or removed in it can be found under the changelog below.

    Changelog:

    • Changed the shortcut's directory name from "MCPE Toolbox" to "Project Lumina." A shortcut reset must be done before updating to v1.2.0!
    • Added a first time shortcut configuration that lets you choose where specific folders are
    • Added a PVP section to the mod menu
    • Added UI Customizer section to the mod menu
    • Added "Support Email" to the connections/links
    • Added “Shortcut Loop” to the shortcut settings
    • Added an option to download addons when enabling or disabling
    • Added two more options to the options editor section in mod menu
    • Added a "World Editor" tab to the main menu, which currently does nothing
    • Moved some packs from the hacks section to the PVP and UI section in mod menu
    • Updated some icons in the UI section in mod menu
    • Updated all of the addons so they are now functional
    • Removed automatic addon downloads
    • Other small bug fixes and improvements
    Download

  • v1.1.3

    This release has tons of fixes and improvements. Although, when installing new updates from the shortcut in this release, it will take you to your browser app and load up linkvertise instead of simply letting you install the new version from within shortcuts. This will be fixed in v1.2.0!

    Changelog:

    • Added auto update back
    • Updated credits and disclaimer
    • Added optional help guide on shortcut setup
    • Fixed the "do you want to continue" after shortcut finished running
    • Fixed the reset in shortcut settings
    • Other small bug fixes and improvements
